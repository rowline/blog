## {{< icon "code" >}} Indie Products & Full-Stack Projects
独立创造，探索效率上限。

### {{< icon "apple" >}} 碎碎念 (Suisuinian) App
*聚焦于 iOS 端的深度个人知识与碎片化信息管理体系。*

* **技术栈**: `SwiftUI`, `CloudKit`, `AI Intent Recognition`
* **架构探索**: 重构了 Universal Vault 架构与统一的 MVVM 范式，实现了 Apple Native 体系下高度定制化的 AI 意图识别 (Intent-Based Linking) 与 Vision Intelligence 后台智能解析。后端服务挂载于 OpenClaw 与 Mac 本地系统，实现代理级异步任务处理。

### {{< icon "location-dot" >}} [Race Log](/racelog/) & [Sports Map](/sports/) 运动数据中心
*将作为铁人三项爱好者的运动轨迹与极客技术深度结合的大屏数据中枢。*

* **技术栈**: `React`, `Hugo`, `Tailwind CSS`, `Mapbox GL`
* **工程实践**: 将独立开发的 Vite + React 交互式地图面板与个人的大铁/马拉松生涯记录本，以无缝沙盒的形式完美嵌入 Hugo 静态网站引擎。实现了跨技术栈的共享全局主题、组件化导航与深浅色原生适配。

### {{< icon "globe" >}} "小鸡小程序"
*微信生态内的高可用、轻量级互动前端。*

* **技术栈**: `WeChat Mini-Program`, `Cloud Functions`
* **业务闭环**: 完全由 AI Agent 辅助 Vibe Coding，单人实现了从立项、交互逻辑、服务端云函数数据打通到合规提审的完整商业侧敏捷闭环。
